JVALRESSPEC
# For each J-coupling constant restraint the following is to be specified:
# IPJR, JPJR, KPJR, LPJR            atom sequence numbers of the real atoms 
#                                   defining the dihedral angle phi that is related 
#                                   to the restrained J-value.
# WJVR                              individual J-value restraint weight factor
#                                   by which the J-value restraining term may 
#                                   be multiplied.
# PJR0                              >=0. In case of a full-harmonic J-value
#                                   restraint (H = 0), PJR0 is the minimum-energy
#                                   J-value; in case of an attractive or repulsive
#                                   half-harmonic J-value restraint (H = +- 1),
#                                   PJR0 is the upper or lower bound, respectively,
#                                   beyond which the restraining force becomes
#                                   non-zero.
# H                        -1,0,1   type of J-value restraint.
#                          -1:      half-harmonic repulsive
#                           0:      full harmonic 
#                           1:      half-harmonic attractive
# PSJR                              phase shift or difference between the dihedral
#                                   angle theta formed by the possibly non-existing
#                                   H-atoms defining the J-coupling and the dihedral 
#                                   angle i-j-k-l formed by the real atoms that
#                                   is related to the J-coupling (in degrees).
#                                   phase shift delta = theta-phi.
# A,B,C                             Karplus parameters a, b and c for the J-coupling
#                                   constant expressed as a function of theta.
# 
#
