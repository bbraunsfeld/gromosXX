Distance Restraints

DISRESSPEC
# DISH  DISC   DISN
  0.1   0.153  0.1
# i  j  k  l  type    i  j  k  l  type    r0    w0    rah
  1  0  0  0  0       10 12 11 13 3       0.2   1.0   0
END

The distance restraint is specified in terms of two virtual atoms. The following virtual atom types are known:
0 explicit atom
1 aliphatic CH1
2 CH1 (aromatic)
3 CH2 (stereospecific)
4 CH2 (non-stereospecific)
5 CH3
6 CH3 (non-stereospecific, Val, Leu)
7 CH1 (aromatic at C(delta) or C(epsilon) in Phe or Tyr
8 NH2
cog centre of geometry
com centre of mass

And r0 is the restraint distance, w0 a weight factor (multiplied by the force constant specified in the input file) and rah specifies the type of restraint:
-1 half harmonic repulsive
0 full harmonic
+1 half harmonic attractive

Perturbed Distance Restraints

PERTDISRESSPEC
# DISH  DISC   DISN
  0.1   0.153  0.1
# i  j  k  l  type    i  j  k  l  type n m   A_r0  A_w0  B_r0   B_w0  rah
  1  0  0  0  0       10 12 11 13 3    1 1    0.2   1.0   0.5    2.0   0
END

With A_r0 and A_w0 the restraint distance and weight factor in state A and B_r0 and B_w0 the corresponding quantities in state B. n and m are the parameters for hidden restraints.
