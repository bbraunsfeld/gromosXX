For position constraints the positions are ignored and in every step the forces and velocities of the constrained atoms set to 0.

B - Factors

The BFACTOR block is optional. If not specified the atomic B-Factors are set to 1. The force constant is given as K = K0 / BFactor, where K0 is specified in the input parameter file.
