Status

Current release: 0.2.1 (24/11/2004) (debian package)

Changes with respect to 0.2.0:

  • documentation updates (html)
  • moved (total) energy calculation to a seperate algorithm
  • remove com motion: NaN fix if det(angular momentum) = 0
  • position constraints (not for SHAKE!)
  • allow empty lines and spaces after block-names
  • TCOUPLE : ir and molecular com bath index interchanged
  • building statically linked executables
Changes with respect to 0.1.3:
  • bugfix: anisotropic pressure coupling
  • fast solvent (spc) loops per default off
  • Grid pairlist algorithm (rectangular box)
  • removed bekker specification
  • replica exchange support
  • fast solvent loops
  • Intel C++ 8.1 support
  • Nose-Hoover temperature coupling
  • Flexible Constraints Bugfix: perturbed / non-perturbed coupling
  • removed Blitz++
  • steepest descent initialization bug
  • removed exceptions
  • changed template instantiation
  • truncated octahedron support
Changes with respect to 0.1.2:
  • Roto-translational constraints
  • Scaling of individual interactions: dihedral angle
  • J-value restraints
  • exact flexible constraints
  • Different lambda values for specific interactions (nonbonded)
  • some Windows support
  • removed size_t variables
  • file name changes tcc -> cc
  • gromosXX scripting example: script added
  • gromosXX scripting support: easier initialization
  • block averages
Changes with respect to 0.1.1:
  • Internal checks on energy, force, energy derivative, pressure, etc.
  • Split up of non-bonded routines
  • Perturbed SHAKE fix
  • Atomic virial fix
  • Different lambda dependencies
  • New library structure
  • Dihedral angle monitoring
  • Open MP
Changes:

CVS access

web cvs access: CVS repository
To check out GromosXX for the first time
  1. setenv CVSROOT :pserver:igc@igc.ethz.ch:/home/cvs/gromosXX
  2. cvs login
  3. password: gromosXX
  4. cvs checkout -P gromosXX
To update your distribution (back up any changes you made first)
  1. cvs update -d -P
